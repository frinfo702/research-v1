# Manuscript

This folder contains the paper/manuscript source for the `research-v1` repository.  
Use this for writing (drafts, revisions, and camera-ready), not slides.

## What goes here

- Main manuscript source (Quarto/LaTeX/Markdown)
- Figures and tables used by the manuscript
- Bibliography files (`.bib`) and citation styles
- Submission assets (optional): cover letter, response to reviewers, etc.

## Recommended layout (maybe changed later)

You may structure this folder like:

- `manuscript/`
  - `src/` — editable sources (`.qmd`, `.tex`, `.md`)
  - `fig/` — figures (SVG/PDF/PNG)
  - `tbl/` — tables or generated table assets
  - `bib/` — references (`references.bib`, CSL files)
  - `template/` — LaTeX templates / partials / Quarto extensions
  - `build/` — generated outputs (PDF), not committed
